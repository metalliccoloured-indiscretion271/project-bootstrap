## 📋 About project-bootstrap

project-bootstrap is a program to help users set up coding projects easily on their Windows computers. It provides a simple way to start new programming tasks without dealing with complex settings. This app organizes files and folders so that projects have the right structure from the start.

You don't need to know how to write code or use command lines. This tool guides you through the initial setup step-by-step.

---

## ⚙️ System Requirements

To run project-bootstrap smoothly, your computer should meet the following:

- Windows 10 or later (64-bit recommended)
- At least 4 GB of RAM
- 500 MB free disk space
- Internet connection for initial download

The program runs offline after setup. It uses little CPU and memory, so it works well even on mid-range computers.

## ▶️ Running project-bootstrap

1. Open the app by double-clicking the shortcut.

2. The program will show a window with options to create a new project.

3. Follow the on-screen prompts:

   - Enter a name for your project.

   - Choose a location on your computer to save it.

4. The program sets up the folder and main files automatically.

5. You can then start adding your own code files or work with other tools.
